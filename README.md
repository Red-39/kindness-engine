# KindnessEngine

KindnessEngine is a sleek, professional web application designed to generate context-aware compliments. Built with a focus on modern UI/UX principles, it uses glassmorphism and smooth CSS transitions to create a calming, positive user experience.

## Features

* **Three Curated Modes**: 
    * **Professional**: Refined feedback for colleagues and workplace scenarios.
    * **Heartfelt**: Deeply sincere affirmations for friends and loved ones.
    * **Silly**: Lighthearted, pun-filled compliments to brighten someone's day.
* **Modern Interface**: Features a glassmorphic card design, blur blobs for depth, and high-quality typography (Inter & Playfair Display).
* **Smart Logic**: Includes a duplicate-prevention algorithm to ensure you don't see the same compliment twice in a row.
* **One-Click Copy**: Integrated clipboard functionality with a custom toast notification.
* **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## Tech Stack

* **HTML5**: Semantic structure.
* **CSS3**: Custom properties (variables), Flexbox, Grid, and keyframe animations.
* **JavaScript (ES6+)**: Modular logic handling state management, DOM manipulation, and clipboard API.
* **Google Fonts**: 'Inter' for UI elements and 'Playfair Display' for the compliment text.

## Project Structure

The codebase is organized into four logical modules:
1.  **DATA REPOSITORY**: A structured object containing 100+ unique compliments.
2.  **STATE MANAGEMENT**: Tracks current categories and animation status.
3.  **CORE UTILITIES**: Pure logic for randomizing items and clipboard interaction.
4.  **UI CONTROLLER**: Manages event listeners, transitions, and DOM updates.

## Usage

1.  Open `index.html` in any modern web browser.
2.  Select a category from the top navigation bar.
3.  Click **Generate Compliment** to trigger a new affirmation.
4.  Use the **Copy Icon** to save the text to your clipboard for sharing.
